# working-title-on-the-same-page-v2
Revamp of "on the same page"-project in C#/.NET

--What is "on the same page"?--
A book club web app representing the status of our book club (current book, members, discussion).
New books (API) can be suggested for a voting poll.
Read books and their discussion can be archived.

--MVP
user registration/login, book api display, voting poll, discussion panel, book rating
